WEB DESIGN

El diseño web es una actividad creativa que se basa en el diseño de plataformas digitales, como un sitio web o una app, creando una experiencia funcional para el usuario ubicando los diferentes elementos de una entidad o empresa.

BOX MODEL

El nombre box- model se define como modelo de caja, porque en los navegadores todo se construye a traves de cajas, cada elemento que definimos en un documento html se muestra en el navegador como una caja rectangular, asi se representan todos los elementos y no existen elementos triangulares, redondos, poligonales etc puesto que internamente el navegador dibuja un rectangulo.

El modelo de cajas o "box model" es seguramente la característica más importante del lenguaje de hojas de estilos CSS, ya que condiciona el diseño de todas las páginas web. El modelo de cajas es el comportamiento de CSS que hace que todos los elementos de las páginas se representen mediante cajas rectangulares.

Las cajas de una página se crean automáticamente. Cada vez que se inserta una etiqueta HTML, se crea una nueva caja rectangular que encierra los contenidos de ese elemento.

MOCKUP

El mockup lo definimos como un fotomontaje de alta calidad, utilizado en el diseño y la publicidad, se puede utilizar en la creacion de un logotipo, pagina web o envase de algun producto, para dar a conocer un posible resultado, es un bosquejo de lo que nuestros clientes solicitan, se indica como quedaria el proyecto y asi el cliente puede solicitar cambios si asi lo desea, sin la necesidad de modificar todo el codigo.
